# Modal 模态框

Title
Content 
<template>
    <div>
        <MantButton @click="visible = true">
            Show Modal
        </MantButton>
        <MantModal
            title="Title"
            :visible="visible"
            @close="visible = false"
        >
            Content
        </MantModal>
    </div>
</template>
<script>
    export default {
        data() {
            return {
                visible: false
            }
        }
    }
</script>

参数 说明 类型 默认值
